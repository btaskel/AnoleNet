# AnoleNet: 一种点对点双方匿名的去中心化通讯系统
AnoleNet: A decentralized communication system with anonymous p2p communication between two parties.

它介绍了一种不使用中心化服务器，可以在不知道双方IP地址的情况下，仅根据'标识'就能与另一台主机进行数据交流的通讯系统。
